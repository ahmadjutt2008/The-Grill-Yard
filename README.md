# The Grill Yard | Premium Restaurant Website

A high-end, responsive restaurant website designed for **The Grill Yard**, featuring modern aesthetics, performance optimizations, and a unique dual-mode user experience.

-   **Premium Design**: Sophisticated charcoal and gold color palette with glassmorphism effects and modern typography.
